## Step 4:
End the program

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: akash m
#RegisterNumber:212223240003
import numpy as np
A=np.array([[6,2,3],[3,1,1],[10,3,4]])
inverse=np.linalg.inv(A)
print(inverse)
```
